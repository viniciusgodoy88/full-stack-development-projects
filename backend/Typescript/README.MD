# 📖 Sobre

Esta pasta contém projetos desenvolvidos para estudo de **TypeScript aplicado ao backend**, explorando conceitos fundamentais da linguagem e da programação orientada a objetos.

Os códigos foram criados com o objetivo de praticar:

- Tipagem estática
- Organização modular
- Classes e Interfaces
- Generics
- Namespaces
- Manipulação de objetos e estruturas de dados

Todos os exemplos foram implementados utilizando **Node.js + TypeScript**.

---

# 🧠 Conceitos Abordados

Durante os estudos nesta pasta foram praticados os seguintes conceitos:

### 🔹 TypeScript

- Tipagem estática

- Encapsulamento

### 🔹 Programação

- Modularização

# 📚 O que foi aprendido

Ao desenvolver os códigos desta pasta foi possível aprender:

✔ Como criar **interfaces e implementá-las em classes**

✔ Como usar **tipos genéricos (Generics)** para tornar classes reutilizáveis

✔ Como organizar projetos utilizando **modules e namespaces**

✔ Como aplicar **POO em TypeScript**

✔ Como criar métodos e manipular dados utilizando tipagem forte

✔ Como estruturar código backend de forma mais organizada
