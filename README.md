This repository contains **hands-on projects and practical exercises** developed during the  
**Postech Full Stack Development** postgraduate program by **FIAP + Alura**.

The goal of this repository is to document learning progress, organize projects by stack area, and build a solid full stack portfolio.

## 🎓 About the Program

- Institution: FIAP + Alura
- Program: Postech – Full Stack Development
- Workload: 360 hours
- Focus: Practical, project-based learning
